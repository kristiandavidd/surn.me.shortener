# 🚀 surn.me — URL Shortener & Link-in-Bio Platform

**surn.me** is a modern web application that provides:
- 🔗 URL shortening (like Bitly)
- 🌳 Link-in-bio pages (like Linktree)
- ⚡ Fast, minimal, and scalable architecture

---

## ✨ Features

### 🔗 URL Shortener
- Generate short URLs (e.g. `surn.me/abc123`)
- Fast redirection
- Optional click analytics

### 🌳 Link-in-Bio
- Create a personal page with multiple links
- Custom username (e.g. `surn.me/f/username`)
- Add, edit, and manage links easily

---

## 🛠️ Tech Stack

- **Frontend & Backend**: Next.js
- **Storage**: Cloudflare R2
- **Database**: Turso
- **Deployment**: Vercel
